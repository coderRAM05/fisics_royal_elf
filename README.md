# **ğŸ•Œ Yantra Dimension Generator: A Parametric Tool for Astronomical Heritage**

The Jantar Mantar Dimension Generator is a Python/Streamlit web application that translates the complex geometric principles of the 18th-century Indian astronomical observatories (Yantras) into precise, scalable architectural blueprints.

This tool solves the challenge of adapting the fixed geometry of the Yantras by generating all required angles and dimensions based on a user-defined Latitude ($\phi$) and Scale Factor ($R$).
